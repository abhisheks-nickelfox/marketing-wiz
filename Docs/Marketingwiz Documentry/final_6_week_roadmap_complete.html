Users → Team page · Dashboard with tabs

Name + avatar + @handle · Status (Active/Invited/Disabled) · Role (job title) · Email · Skills tags · Extra Permissions popup (Project Creation, Task Creation, Global Timesheet) · Row: Settings gear, Delete, Edit · Pagination (100 users)

Week 4

Project Summary + Task Management + My Tasks

Projects global view · Project detail · Activity panel · My Tasks personal view

Project Summary ✅

Grouped by: To Do, Assigned, In Progress, Internal Review · Columns: name + task type tags, client, assignee, due date, priority, status · Row actions: Edit, Delete (with safety), Convert to Template, Export project data · Filter: 12 status types + Firms

Project Detail ✅

Assignee, Status (with → flow), Priority, Due date · Task Type tags · Timesheet (2hr 30m + play → time logging panel) · Description · Attachments (PDF + MP4) · Custom Fields (Service type) · Sub Tasks · Actions: Edit, Delete (with safety), Convert to Template

Time Logging Panel ✅

Manual time entry OR start timer · Date/time range · Notes · Billable toggle · Save · Time entries per user (expandable)

Activity Panel ✅

Recent / Files & Links / Notes tabs · Chat thread (messages, file attachments, read receipts) · Emoji reactions popup · Message input + send

My Tasks ✅

Task Assigned to you · Left sidebar: To do, Assigned to me, Today Due, Overdue, Active, Assigned, In Progress, Urgent, Blocked, Revisions, Closed, Complete (with counts) · Task tooltip (Project, Client, Assigned by) · Filter: full status list + Firm
Custom Fields builder — display only, no creation UI designed
Week 5

Inbox + My Timesheet

Inbox · My Timesheet personal weekly grid

Inbox ✅

Grouped: Today / Yesterday / Last 7 Days / Monthly · Mark all as read, Clear all, Filter · Click item → right panel (thread, @mentions, file attachments, status changes, assignment changes) · Reply, emoji react · Filter: Mentions (4), Replies (2), Unread (6), Assigned to me (11), Overdue (8), Cleared (5), Clients list with search · Clear / Apply filter

My Timesheet ✅

Personal weekly grid view (Sun–Sat) · Date navigation (prev/next week) · Rows = tasks · Cells = hours logged per day with clock icon · Total column · + Add task button · Global header timer (02h:30m:25s + pause)
Week 6

Time Reports (By Team + By Client) + Team Pulse

Time reports → By Team · By Client · Team Pulse real-time monitoring

Time Reports — By Team ✅

Team members table: Name, Hours Spent, Allowed Hours, Hourly Rate · Daily/Weekly/Monthly/Select dates · 100 users paginated · Click member → right panel: role badge, time period selector, Sort by Team + Status, Billable toggle, Total Hours (spent/allowed), Hours allocation by firm (expandable with task + status badge), Export timesheet button

Time Reports — By Client ✅

Firms/Customers table: Company, Accounts Manager (avatar + name), Primary Contact (avatar + name), Time Spent · "These companies have purchased in the last 12 months" · Click firm → right panel: firm name, Daily/Weekly/Monthly, Sort by Firm + Status, Billable toggle, Total Hours, Hours by project (expandable: project name + hours), Export timesheet button

Team Pulse ✅

Daily/Weekly/Monthly/Select dates · Table: Name, Total Tasks (count + info icon), Pulse status badge (Available Soon / Available) · Info icon tooltip → Task by status breakdown (To Do, Assigned, Inprogress, Revisions, Blocked, Internal Review, Client Review, Completed with counts) · 100 users paginated
